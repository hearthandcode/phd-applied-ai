### Format observations

Reading-to-recall gap is the primary format challenge: the learner comprehends while reading but retention is poor without active encoding (exercises, note-taking, re-explanation). Multiple return visits required for the same passage. This is consistent with working memory load profile, not comprehension failure.

Visual-spatial intuition is the learner's native channel for abstract structures. The unprompted description of functions as "floating boxes with inputs and outputs" maps directly onto categorical/morphism thinking — a strong cognitive anchor for the dependency-map sections. Formal notation (Greek letters, set notation) functions as a translation overhead, not a primary representation.

No computational exercises were attempted this session. The Python/NumPy blocks in Checkpoint 1 are the correct next step; they will close the loop between conceptual understanding and active encoding.
